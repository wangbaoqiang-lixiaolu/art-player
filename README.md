安装教程：
安装环境：PHP7.0+MySQL5.6
下载源码解压至服务器根目录，需要在根目录，暂不支持二三级目录
弹幕库安装访问/dmku/install,填写数据库信息进行安装
安装完成访问你的域名+/admin 登录管理后台，初始用户名密码为下载安装包文件内查看
完成安装播放视频：你的域名+/?url=视频地址
如官方视频等无法解析，可登录播放器后台修改检查您的 JSON 是否可以解析以及是否是标准接口
可添加多条备用 json 接口，当主接口出现问题自动切换备用接口）
